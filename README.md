## ✨ Features

- 🎭 Interactive Easter scroll (RO + EN)
- 🌑 Cinematic dark mode UI
- 🥚 Egg cracking mini-game ("knock-knock" effect)
- 🎰 Rarity system: **common / rare / legendary**
- ✨ Golden Egg moment
- 🔥 Combo system
- ⏳ Live countdown to Easter
- 🌅 **Easter Day Mode** (auto-activates on April 12)
- 🕯️ Candle ritual interaction
- 🌌 Easter constellation with hidden messages
- 🎬 Multiple endings based on choices
- 📱 Mobile-first design
- ⚡ Zero dependencies (pure HTML/CSS/JS)

## 🚀 Deploy on GitHub Pages

1. Create a new repository  
2. Add these files:
   - `index.html`
   - `favicon.svg`
   - `preview.png` (recommended)
3. Push your code  
4. Go to **Settings → Pages**  
5. Select branch → `/root`  
6. Save
